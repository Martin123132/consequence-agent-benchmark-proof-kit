# Proof Scope

This proof kit is a public evidence wrapper around a private benchmark engine.

It is designed to prove that CAB exists, has a working release evidence story,
has validated benchmark families, and has a checksum-indexed local leaderboard
workflow worth evaluating. It is not designed to reveal implementation internals.

## Public In This Repository

- product description
- public-safe evidence-card summaries
- benchmark score headlines
- release-readiness summary
- local leaderboard proof summary
- access request workflow
- commercial boundary
- public-materials licence

## Not Public In This Repository

- engine source code
- package internals
- private release wheels or source distributions
- benchmark implementation details
- private benchmark threshold configuration
- unpublished notebooks
- private GitHub workflow details
- private customer or pilot material

## Why This Boundary Exists

The public proof kit gives reviewers enough to evaluate the shape of the
technology without turning the engine into a public code drop.

Commercial use, private repo access, and implementation review must happen
through written permission.
